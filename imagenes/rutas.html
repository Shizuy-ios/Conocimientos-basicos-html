Rutas en HTML

Introducción

Hay dos tipos de rutas: relativa y absoluta.

Ruta relativa: accede a archivos de la misma carpeta usando solo el nombre, o con "../" para retroceder carpetas según la ubicación del archivo.

Ruta absoluta: apunta siempre a la misma ubicación, sin importar dónde esté el archivo actual.

Cómo moverse entre carpetas con rutas relativas

Para acceder a un archivo en una carpeta anterior: si la página "rutas" está en la carpeta imagenes y quieres enlazar a Listas.html, escribes:

<a href="../Listas.html">Listas</a> Listas (correcto)

Si escribes solo "Listas.html" sin "../", el enlace no funciona:

Listas (incorrecto)

Para acceder a un archivo en una carpeta que está más adelante:

Escribes el nombre de la carpeta seguido del archivo, por ejemplo:

<a href="ejemplo/ejemplo1.html">Ejemplo 1</a> Ejemplo 1

Resumen

Para retroceder una carpeta: ../
Para retroceder dos: ../../
Para entrar a una carpeta: (nombre_de_carpeta)/
Para entrar a carpetas consecutivas: carpeta1/carpeta2/
